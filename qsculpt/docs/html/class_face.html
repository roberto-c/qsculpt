Detailed Description

Face class. This class contains references to points that should form a triangle.

The points are indices of the Object3D point list. So, this class It's only meant to be used with the Object3D class (its not a generic triangle class).

A triangle its only valid if theirs three points are different. Using the default constructor makes a non valid triangle.
